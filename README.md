# minipad

minipad -- a simple editor, inspired by [kilo](https://viewsourcecode.org/snaptoken/kilo/)

### Execute
`./build/minipad <filename>` 

or

`./build/minipad` 

### Compile
`cd minipad/`

`make`

  Makefile supports:
    `make`
    `make debug`
    `make clean`
